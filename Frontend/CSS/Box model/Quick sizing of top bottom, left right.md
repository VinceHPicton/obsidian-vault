This does padding 1em for top/bottom and zero for left/right
```
.container {
	padding: 1em 0;
}
```

This does padding 2em for left/right and zero for top/bottom
```
.container {
	padding: 0 2em;
}
```

This goes around like a **clock**: 
1. top: 0
2. right: 2em
3. bottom: 1em
4. left: 3em
```
.container {
	padding: 0 2em 1em 3em;
}
```
